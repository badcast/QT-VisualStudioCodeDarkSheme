# Install

## Windows
`xcopy VisualStudioDarkX.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp VisualStudioDarkX.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp VisualStudioDarkX.xml ~/.config/QtProject/qtcreator/styles/`
